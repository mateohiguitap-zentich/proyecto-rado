MANUAL DE USUARIO: MÓDULOS PRINCIPALES

SISTEMA FACTURACIÓN RADO

Este manual servirá de guía para que el usuario pueda utilizar de manera óptima nuestra plataforma web.

Índice Temático

Haz clic en cualquier tema para ir directamente a su explicación:

  1. Módulo Panel de Control
  2. Módulo Registro de Paciente
  3. Módulo Orden Virtual
  4. Módulo Factura Electrónica
  5. Módulo Generación de Reportes

1. Módulo Panel de Control

En este módulo se encontrará un resumen del día dependiendo de la sede en donde estemos ubicados.

2. Módulo Registro de Paciente

En este módulo se empieza a dar funcionamiento a la plataforma. Primeramente, iniciamos digitando el número de documento del paciente.

  • Si el documento digitado ya existe, el sistema mostrará una ventana emergente donde el usuario cargará la información asociada a ese número de documento ingresado.
  • De no estar registrado el número de documento, el usuario podrá pasar por cada uno de los datos que debe llenar usando el mouse o la tecla TAB. Es obligatorio llenar cada uno de los datos; de hacer falta alguno, el sistema mostrará un mensaje notificando qué datos hacen falta. En caso tal de que el paciente no desee registrar algún dato, el usuario pondrá NA.

Una vez finalizada la captación de datos, el usuario deberá dar clic en el botón guardar que se encuentra en la parte superior derecha. Para luego seguir con el proceso de generación de orden virtual dando clic en el botón “FINALIZAR REGISTRO Y GENERAR ORDEN VIRTUAL”.

3. Módulo Orden Virtual

El sistema cargará la información necesaria del paciente, encontrada en la sección “Datos del Paciente”.

El usuario deberá llenar la sección de “Información del Servicio”, donde podrá agregar uno o más estudios a realizar, odontólogo remitente y su correo electrónico (en caso tal de no tener, se pone NA).

En Detalles de entrega, se selecciona el “Formato de entrega”: FÍSICO, VIRTUAL o FÍSICO/VIRTUAL.

Una vez finalizada la sección, el usuario debe dar clic en “Guardar Orden” para poder enviar la orden en formato PDF a los técnicos, además servirá como puente para la generación de la factura electrónica.

4. Módulo Factura Electrónica

El sistema cargará la información de factura, información de contacto, información de la clínica (según la sede donde estemos facturando) e información de facturación.

El usuario deberá diligenciar el valor a pagar y el método de pago (EFECTIVO, CONSIGNACIÓN/TRANSFERENCIA, TARJETA DÉBITO O TARJETA CRÉDITO).

Una vez diligenciado todo correctamente, el usuario debe dar clic en guardar y luego enviar, botones que se encuentran en la parte superior derecha de la pantalla.

5. Módulo Generación de Reportes

Se podrá evidenciar por el momento: CONSOLIDADO POR SEDE, FACTURACIÓN POR SEDE y AGENDAMIENTO DE CITAS. En un futuro se puede agregar otro tipo de reporte según las necesidades del cliente.

  • La fecha de inicio debe ser inferior o igual a la fecha final.
  • Previsualizar: Es el botón donde el usuario podrá obtener una vista previa del informe previamente solicitado.
  • Exportar: En este botón el sistema genera un archivo en Excel, según el informe solicitado.

6. Módulo Gestión de Usuario

En este módulo solo podrá tener acceso el personal administrativo autorizado. Acá se podrá agregar un nuevo usuario, el cual debe tener número de documento, clave única y rol asignado. Si el número digitado ya se encuentra en uso, el sistema no dejará guardar la información.

  • Editar usuario: El usuario debe ingresar el número del documento que desea editar, donde podrá cambiar la clave y asignar un nuevo rol. Si el número de documento ingresado no existe, no generará ninguna acción.
  • Eliminar usuario: El usuario podrá eliminar un usuario existente, ingresando su número de documento y validando en el sistema que exista para que posteriormente el sistema muestre una ventana confirmando que se desea eliminar al usuario seleccionado.
